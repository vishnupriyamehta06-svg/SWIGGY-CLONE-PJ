# 🍔 SWIGGY – Swiggy Inspired Homepage Clone!!

A modern **food delivery homepage UI** inspired by popular food-tech platforms like Swiggy.  
This project is built using **pure HTML and CSS** for learning and frontend practice purposes.

The goal of this project is to understand layout structuring, responsive design, and modern UI patterns used in food delivery platforms.

---

## 🚀 Project Overview !!

FoodNest is a static homepage that simulates a multi-service food platform experience including:

- Food ordering section
- Grocery delivery section (Instamart-style)
- Restaurant discovery section (Dineout-style)
- Service highlights
- Promotional banner
- App download section with QR code
- Footer with company and support links

This project focuses on **UI recreation and frontend fundamentals**, not backend functionality.

---

## ✨ Features !!

✅ Sticky navigation bar  
✅ Hero section with search interface  
✅ Food category grid (10+ items)  
✅ Grocery delivery grid (10+ items)  
✅ Restaurant discovery cards  
✅ Service sections (Quick Delivery, Groceries, Dineout)  
✅ QR code app download section  
✅ City-based delivery information  
✅ Responsive layout using CSS Grid & Flexbox  
✅ Smooth hover effects and modern UI styling  

---

## 🛠️ Technologies Used !!
